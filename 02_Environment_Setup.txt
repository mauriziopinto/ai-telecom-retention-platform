Local Machine Instructions
Before you begin the project, make sure your environment is set up correctly. This will help you avoid errors and ensure your integrative industry synthesis artifact, notebook, or script runs smoothly from start to finish.

Install Python 3.x You can use any recent version of Python 3 (3.9 or higher).
Install a development environment You may use any of the following:
Jupyter Notebook
JupyterLab
VS Code with the Jupyter Extension
Anaconda (includes Jupyter by default)
Install required Python libraries Make sure the following core libraries are installed:
NumPy
Pandas
Matplotlib Depending on the scope of your industry synthesis, you may also use machine learning, deep learning, generative AI, or agentic AI libraries that were explicitly covered in earlier capstone projects and course materials. No new technologies outside the approved coursework are required. If you are using Anaconda, many of these libraries may already be available.
Validate Your Environment
Open a new Jupyter Notebook or Python script.
Run the following code to confirm core libraries import correctly:
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
If your synthesis artifact includes models, agents, or workflows from earlier projects, run a minimal test cell or script that executes a representative component (for example, loading a model, running a pipeline step, or executing an agent action).
Confirm that all components execute without errors and produce expected outputs.
If the imports run without errors and your test execution succeeds, your environment is ready.

Create Your requirements.txt File
Before submitting your project, generate a reproducibility file from your environment:

   pip freeze > requirements.txt
This file allows others to rerun your integrative industry synthesis using the same package versions.
